# Homemade LLM

A small transformer-based language model built for **learning and experimentation**.

This project is designed to help you understand, from the ground up, how an LLM is:
- trained
- making predictions
- improving over time

It is intentionally simple, slow, and fully readable so you can follow every step of the process.

It was originally designed to be trained locally on a laptop.

---

## 🎯 Purpose

This repository exists for **study purposes**.

Instead of using large, complex frameworks, this project shows:
- how a language model actually works internally
- how training changes its behavior
- how predictions are formed step by step

Think of it as a **mini laboratory for LLMs**, where you can observe and experiment with:
- probabilities of next-token predictions
- attention mechanisms
- loss reduction during training
