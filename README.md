# Homemade LLM

A small transformer-based language model prototype built for study and visual debugging.

## Setup

```powershell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run

```powershell
python tiny_llm_visual_debug.py
```

## Notes

- Edit `training_text.py` to change the text corpus used for training.
- The project is intentionally small and readable so it can be studied from the ground up.
